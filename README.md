# Flow State

フレームボタンをクリックするだけで、集中モード（フロー状態）のオン・オフを切り替えるシンプルなUIアプリ。

フロー状態に入ると画面が暗転し、炎がゆらめき、オレンジのグロウが広がる。視覚的な変化が「今は集中している」という感覚を強化する。

## Stack

- Next.js (App Router)
- Hono (type-safe RPC)
- Framer Motion
- Tailwind CSS v4
